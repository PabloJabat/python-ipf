# python-ipf

Iterative proportional fiting implementation in Python 3.9.0

## Installation

```bash
python -m venv .venv
source .venv/bin/activate
pip install -r requirements.txt
```

## Usage

```bash
source .venv/bin/activate
python -m ipf
```
